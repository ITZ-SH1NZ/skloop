## Vision and Core Philosophy

Skloop is not just a learning management system; it is a **Habit Engineering Platform**. We believe the biggest obstacle to technical mastery is the **Motivation Gap**.

Skloop transforms the अक्सर-sterile process of learning to code into a rich, visceral, and social adventure. Our core philosophy—**"Consistency Beats Intensity"**—is baked into every pixel of the UI.

---

## The Roadmap Ecosystem

Navigate technical knowledge through two distinct, high-fidelity experiences:

### The Gamified Path (The Journey)
An interactive "Level Map" that winds through three distinct visual biomes:

- <img src="https://img.shields.io/badge/Grasslands-D4F268?style=flat-square&logoColor=000000" alt="Grasslands" /> **Beginner**: Foundations of the path.
- <img src="https://img.shields.io/badge/Mystical_Forest-2D5A27?style=flat-square&logoColor=FFFFFF" alt="Mystical Forest" /> **Intermediate**: Deepening technical roots.
- <img src="https://img.shields.io/badge/Crystal_Caves-70D6FF?style=flat-square&logoColor=000000" alt="Crystal Caves" /> **Advanced**: Mastering the core.

### The AI Technical Graph (The Constellation)
A dynamic, non-linear view for specialized goals, generated by our AI engine:
- **Topological Sorting**: Mathematical depth-based arrangement.
- **Animated Edges**: Bezier curves with flowing "data packets" visualizing prerequisites.

## Core Features

- **Mission Control (Dashboard)**: Central nervous system with "Today's Focus," GitHub-style consistency heatmaps, and real-time leaderboards.
- **Deep-Dive Tracks**:
  - **Web Development**: Mastering the "Porcelain" standard—high-end UI/UX with modern architectures.
  - **DSA (The Logic Core)**: Pattern recognition and algorithmic foundations.
- **Specialized Dev Tools**: Integrated Interactive IDE and a drag-and-drop Flowchart Builder.
- **Practice Arena**: Daily mini-games like **Codele** (technical Wordle), Typing Speed Tests, and DSA Quizzes.
- **Social Layer**: Study Circles (Squads), Peer Connections, and Global/Friends Leaderboards.
- **Mentorship Ecosystem**: Find expert "Guides" through a discovery portal and access exclusive instructional content.

## Technical Stack

Built with a commitment to visual excellence and low-latency performance:

- **Frontend**: Next.js (App Router), Tailwind CSS, Framer Motion, GSAP.
- **Real-Time Data**: Supabase (PostgreSQL, Realtime, Auth, Storage).
- **Core AI**: Groq (Llama 3.3 70B).

## Getting Started

### Prerequisites
- Node.js 18+
- npm / yarn / pnpm

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/skloop.git
   cd skloop
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env.local` file based on the required Supabase and Groq keys.

4. **Run the development server**:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to start your journey.
